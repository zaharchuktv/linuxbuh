           Разноска выписки банка.

	Если ваша организация пользуется электронной системой платежей
"Банк-Клиент", то для разноски выписки банка вам нужно
обратиться к программистам вашего банка, чтобы они сформировали
файл с випиской по вашему счету в формате, приведенном ниже:

   Необходимый формат записи выписки из банка для загрузки
                данных в систему "iceB".

Приход/расход (+/-)        |+
Дата документа             |1.2.2002
Номер Документа            |458
Сумма                      |1000.15
Код ЕДРПУО                 |0195623
МФО                        |802015
Счет                       |123456789
Наименование контрагента   |кненоеглкнглкншдешдегшд
Назначение платежа         |фвеоырвполаробарлполюпол
Счет получателя/плательщика|123365644
Счёт,счёт корреспондент    |
Код контрагента            |

Приход/расход (+/-)        |-
Дата документа             |1.2.2002
Номер Документа            |567
Сумма                      |345.15
Код ЕДРПУО                 |4567823
МФО                        |802015
Счет                       |123456789
Наименование контрагента   |кненоеглкнглкншдешдегшд
Назначение платежа         |фвеоырвполаробарлполюпол
Счет получателя/плательщика|123365644
Счёт,счёт корреспондент    |
Код контрагента            |

  Последние два реквизита в каждом блоке не обязательны для заполнения. 
Если эти реквизиты будут заполнены, то программа загрузит всё автоматически.
Если эти реквизиты не будут заполнены, то программа будет показывать на
экран каждый блок для заполнения оператором этих реквизитов.
"Счёт,счёт корреспондент" вводятся через запятую. Если будет введён только 
один счёт, то программа будет считать, что это счёт корреспондент. Счёт она 
определит по номеру расчётного счёта из файла настройки выполнения проводок
для подсистемы "Платёжные документы".


	После входа в этот режим на экране появиться меню:

-------------------
|Введите имя файла|
|                 |
-------------------

F1 помощь F2/+ загруз. F3 просмотр F10 выход

F1  - помощь; 
F2  - начать загрузку данных;
+   - начать загрузку данных;
F3  - просмотр файла
F10 - выход.

  	Если вы, не вводя имени файла, нажмете Enter,
то программа покажет список файлов, в каталоге из которого была
запущена программа и вы сможете выбрать нужный вам
файл, в котором находится выписка.
        Если имя файла введено, то, нажав клавишу F3 вы сможете просмотреть
его содержимое.
