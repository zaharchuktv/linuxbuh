               Программа для автоматического списания розницы.

	Программа "fsprozn" предназначена для автоматического списания
розничных продаж со складов. Если вы эксплуатируете программу "rozn", то
каждый день после завершения торговли вы должны выполнить процедуру
списания проданного товара с карточек материалов.
	Это можно сделать двумя способами. Зайти в программу "xmu" и
запустить там соответствующий режим или воспользоваться программой
"fsprozn". Запуск этой программы можно настроить в
автоматическом режиме. Она спишет всю реализацию которая была сделана до
даты запуска программы, включая и дату запуска программы.
        Программа должна запускаться с ключами передающими ей
пароль для доступа к базе данных и именем базы данных.
        Например:
fsprozn -p aaa -b bx
            |     |
            |     -> Имя базы данных
            -> Пароль
        Протокол работы программы будет выгружен в файл /tmp/iceb_srммгггг.pr
"мм" в имени файла означает месяц запуска программы
"гггг" - год запуска программы.

	Необходимо помнить что если вы настроили автоматическое списание,
то реализация должна всегда заканчиваться до запуска программы и машина не
должна быть выключена до запуска программы. Если сервер работает в
круглосуточном режиме, то лучше всего запуск программы настраивать
на 0 часов (полночь). При условии что ночью никто не торгует. Если магазин
работает круглосуточно, то списание реализации должно выполняться только в
ручном режиме.
