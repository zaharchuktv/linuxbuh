                 Загрузка списка материалов.

	Зтот режим применяется для первоначальной загрузки данных в
справочник товаров и материалов. Данные будут загружатся из файла имя
которого введет оператор.
	После входа в этот режим на экране появится меню с предложением
ввести имя файла в котором находится список товаров, который нужно
загрузить. В нижней части экрана строка подсказки:

F1помощь F2/+ загрузка F3 просмотр F10 выход

F1  - получение описания работы режима;
F2  - начать загрузку данных;
+   - начать загрузку данных;
F3  - просмотрет файл имя которого оператор ввел в меню; 
F10 - выход.

	Формат записи файла должен иметь следующий вид:

123|2|Асфальтобетон|0|0.|Кг|0|0|
 |  |       |       | |  |  | |
 |  |       |       | |  |  | -> Фасовка
 |  |       |       | |  |  -> Кратность
 |  |       |       | |  -> Единица измерения
 |  |       |       | -> Цена (цена по которой предприятие реализует товар)
 |  |       |       -> Код тары
 |  |       -> Наименование товара или материалла
 |  -> Код группы 
 -> Код материалла

	Если вместо кода материала ввести слово "авто", то программа сама
присвоит коды материаллам.
