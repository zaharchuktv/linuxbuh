                 Программа расчета зароботной
                          платы

        Программа выполняет все виды расчетов необходимых для
формирования начислений и удержаний на каждого работника.

        По результатам расчета можно получить:
  ведомость получения зарплаты;
  распечатку начислений и удержаний по категориям работников;
  список работников;
  справки по зарплате за любой период времени;
  реестр почтовых переводов для алиментов;
  список имеющих определенное начисление или удержание;
  список не имеющих определенное начисление или удержание;
  ведомость производственных затрат;
  справку по совместителям для налоговой инспекции;
  список должников;
  перерасчет подоходного налога за год;
  свод начислений и удержаний по счетам;
  корешки с отображением расчета зарплаты для каждого работника;
  свод отработанного времени по категориям;
  свод начислений и удержаний по фамилиям;
  справку за квартал (форма 1дф) для сдачи в налоговую инспекцию;
  кассовый ордер для получения зарплаты.

       Во время расчета зарплаты программа выполняет
все необходимые проводки которые попадают в базу данных
"Главная книга".

        Перед началом эксплуатации системы, необходимо ввести
нормативно-справочную информацию в которую входят:
перечень подразделений;
перечень категорий работников;
перечень начислений;
перечень удержаний;
перечень званий;
перечень видов табеля;
стартовое сальдо по каждому работнику;
перечень городов налоговых инспекций (для иногородних
работников).

        После ввода нормативно-справочной информации необходимо
ввести карточки для каждого работника и сформировать стандартный
набор начислений и удержаний. После расчета зарплаты для
конкретного человека программа выполнит все необходимые проводки
в соответствии с теми счетами, которые были введены в
начислениях и удержаниях. Проводки попадают в базу данных
программы "Главная книга".
        С помощью специального режима программа может
осуществить контроль расчета зарплаты. Программа найдет всех
работников для которых забыли расчитать зарплату, не ввели
табель, не правильно расчитали удержания.
