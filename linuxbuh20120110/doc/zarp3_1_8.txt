         Ввести календарь.

	Этот режим позволяет ввести количество рабочих дней и
часов для каждого месяца. Пограмма будет работать в файле с
помощью текстового редактора который вы используете для работы.
Файл имеет следующий вид:

Календарь рабочих дней и часов
---------------------
Месяц/год|Дней|Часов|
---------------------
8.1998   |21  |168
9.1998   |22  |176

        Эта информация используется программой только для
автоматизированного расчета начислений. Поэтому в файле сохраняется информация только за
текущий год. Остальное можно безболезненно удалять.
