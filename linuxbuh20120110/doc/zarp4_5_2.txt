           Распечатать свод начислений и удержаний
             по счетам в карточках работников.


---------------------------------
|Дата начала............        |
|Дата конца.............        |
|Код подразделения.(,,).        |
---------------------------------

F1 помощь F2/+ расчет F3 реквизиты F4 очистить F5 ком. расч. F10 выход

F1  - помощь;
F2  - расчет;
+   - расчет;
F3  - просмотр списка подразделений;
F4  - очистить меню;
F5  - комплексный расчет с использованием настроек введенных в файл;
F10 - выход.


	Обязательным для заполнения является только первый
реквизит. Если "Дата конца" не введена, то программа
устанавливает ее равной "Дате начала".
	Для заполения меню оператор может нажать клавишу F3 и
выбрав список нужного ему реквизита войти в режим просмотра
выбранного списка. Если оператор находясь в списке нажмет
клавишу Enter, то программа вставит код на котором была подсветка
строки в меню.
	Если оператор войдет в режим "Просмотр списка настроек для
комплексного расчета." программа запустит текстовой редактор и
вы попадете в файл который должен иметь вид:

Заводоуправление:
Подразделения|1,2,3

Бухгалтерия:
Подразделения|5,6,7

	После наименования расчета, который оканчиваеться
двоеточием идет список реквизитов для настройки расчета.
Количетсво наименований расчета не ограничено. 
	Расчет по введенным в файл настройкам осуществляется
нажатием клавиши F5, при этом все реквизиты меню, кроме первых
двух игнорируются.

	Если оператор войдет в режим "Просмотр списка настроек
счетов для расчета." программа запустит текстовой редактор и вы
попадете в файл который должен иметь вид:

Проценты|37,12,88
Счета|41,87
Счета не входящие в расчет 12 процентов|
Счета не входящие в расчет 88 процентов|
Процент|9
Счет|20
	В этом файле насраивается необходимый набор счетов для
расчета отчислений на соц. страх.

	Если оператор войдет в режим "Просмотр списка настроек
субсчетов для расчета." программа запустит текстовой редактор и вы
попадете в файл который должен иметь вид:

Список счетов с субсчетами для расчета свода по балансовым счетам.
80|801,802
81|811

	В єтом файле ввоится список всех имеющихся у счета
субсчетов.
