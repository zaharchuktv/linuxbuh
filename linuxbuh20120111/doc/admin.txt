            Программа администрирования баз данных i_admint

 Эта программа предназначена для аднинитрирования баз данных системы
бухгалтерского учёта iceB. После запуска программы на экране появиться меню:

-----------------------------------------
|Введите логин................          |
|Хост на базу.................          |
|Кодировка запросов к базе....          |
|Путь на настроечные файлы....          |
-----------------------------------------

F2/+ ввод F4 очистить F10 выход

F2  - продолжить работу с программой
+   - продолжить работу с программой
F4  - очистить меню от введённой информации
F10 - завершить работу с программой


 Логин нужно вводить root или другой логин имеющий такие же полномочия как
root. Хост на базу должен быть localhost если база находится на том же
компъютере на котором вы работаете или доступный вам хост на другой
компъютер к базе которого вы имеете доступ. Кодировка запросов должна
соотвествовать той кодировке в которой работает ваш компъютер. Путь на
настречные файлы это пут на каталог в котором должны лежать файлы настройки 
для размеченных баз данных.
 После нажатия клавиши F2 на экране появиться меню

------------------------------------------------
|Введите пароль для доступа к базе данных mysql|
------------------------------------------------
|                                              |
------------------------------------------------

 Нужно ввести пароль который назначен введённому вами логину для доступа к
базе данных mysql. Если пароль введён правильно, то на экране появиться
меню:

----------------------------------------------------
|Разметить новую базу данных для iceB              |
|Разрешить доступ к базе данных                    |
|Просмотр списка разрешённых доступов к базе данных|
|Просмотр списка существующих баз                  |
|Преобразование баз для новой версии системы       |
----------------------------------------------------

 После разметки новой базы данных необходимо всем операторам, которые будут
работать с этой базой разрешить доступ к этой базе данных. Если вам нужно
запретить доступ к базе данных для определённого оператора, то нужно зайти в
режим работы "Просмотр списка разрешённых доступов к базе данных" и удалить
соответствующюю запись.
 После разрешения доступа к базе данных оператор может работать со всеми
подсистемами iceB, блокировать и редактировать файлы настройки. Если нужно
запретить оператору выполнение этих операций, то нужно зайти в режим
"Просмотра списка существующих баз", выбрать нужную базу и нажать клавишу
F6. Вы перейдёте в режим работы с установкой разрешений для работы с
подсистемами iceB.
