                Товарно-материальный учет.

	Система предназначена для автоматизированного ведения
учета любых видов материалов, товаров, малоценки и прочего.
        Система позволяет выписывать и распечатывать накладные и
налоговые накладные на приход и расход любых материалов. При
выписке материалов оператор в режиме реального времени видит
остатки материалов на складах. Это позволяет выписывать
только то, что реально имеется в наличии. Учет ведется для всех
видов материалов (продукция, малоценка, товар и др.).
        После ввода и распечатки накладной оператор имеет
возможность подтвердить выдачу материала по этой накладной либо
полностью, либо частично. После подтверждения накладной оператор
может выполнить проводки к этой накладной либо в ручном, либо в
автоматическом режиме.
        Оператор имеет возможность получить список всех накладных
за любой период времени, либо список только неподвержденных
накладных, либо список накладных, к которым не выполнены
проводки.
     Для начала эксплуатации программы нужно  ввести  
нормативно-справочную информацию. Перечень складов, виды расходов и
приходов, группы материалов (нефтепродукты, строительные
материалы и т.д.), перечень материалов (Бензин А76,Бензин
А93,Цемент,Кирпич и т.д.), перечень единиц измерения и т.д.
     Важным моментом является то, что в списке кодов
контрагентов обязательно должен быть введен контрагент, который
эксплуатирует систему, с кодом "00".  По этому коду должны
проводится все внутренние перемещения материалов.  Если же
пользователь при внутреннем перемещении и виде операции "расход"
укажет код "00-1", где "1" - код склада, то система автоматически
сформирует, кроме документа на расход, документ на приход, для
склада с кодом "1". Код склада должен соответствовать одному из
кодов складов, введенных в систему.
     После этого нужно ввести стартовые  остатки  материалов по
карточкам на момент начала эксплуатации системы. Стартовые
остатки вводятся приходным документом номер которого должен быть
"000", код операции "ОСТ" (стартовый остаток), код контрагента
"00".
     После ввода стартовых остатков все документы на  приход и
расход материалов необходимо проводить через систему. Программа
на основании этих документов будет автоматически проводить
необходимые записи в карточках на приход и расход материала
после того, как документ подтвержден.
        При вводе прихода материала вводится его цена, единица
измерения, счет учета, кратность, цена реализации. Если
комбинация перечисленных реквизитов уникальная, то заводится
новая карточка на материал. Если такая карточка уже есть, то
приход оформляется в существующую карточку.
        При вводе расхода программа показывает все имеющиеся в
наличии карточки на указанный материал с остатками на них.
Оператор выбирает, с какой карточки будет осуществлено списание
материала.
     После введения документа в систему, необходимо подтвердить
этот документ. Бывает так, что документ выписан, а материал по
нему не получен. Поэтому подтверждение документа выполняется
после фактической выдачи/получения материала. Подтверждение
документа может быть полным либо частичным. Система позволяет
получить список неподтвержденных документов, а также документов
по которым проводки не выполнены.
        При выписке документов на расход программа показывает
два остатка: физичесткий и по документам. Физичесткий остаток -
это реальный остаток материала на складе. Остаток по документам -
это остаток, который получится, если все выписанные документы
будут подтверждены, то есть по ним будет выдан/получен материал.
Оператор имеет возможность заказать получение остатков на любую
дату.
     Проводки выполняются только для подтвержденных документов.
Система проверяет правильность проводок по счету учета, по которому
числится материал. Перед началом проведения проводок на экран
выводиться список счетов с суммами, по которым нужно сделать
проводки. Если на экран выведены нулевые суммы, это значит что
документ не подтвержден. Нужно подтвердить документ и только
после этого делать проводки.
     Предусмотрена автоматическая установка номера  документа.
Номер документа формируется из кода склада и порядкового номера
на расход или приход с начала года. Пользователь может
использовать отличную от этой систему нумерации документов. В
этом случае номер документа он должен ввести сам. Программа
сделает проверку на отсутствие такого номера, и если он уже есть -
выдаст об этом сообщение.
     Если пользователь не помнит код материала, то  для  его
поиска достаточно ввести несколько букв из его наименования.
Система выдаст на экран список всех материалов, имеющих данную
комбинацию букв, и пользователь может выбрать нужный ему материал
и подтведить его клавишей "Enter".
