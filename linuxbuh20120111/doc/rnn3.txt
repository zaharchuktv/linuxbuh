           Распечатка реестра налоговых накладных.

	После входа в этот режим на экране появиться меню:



------------------------------------------------------
|Дата начала.(д.м.г)....................             |     
|Дата конца..(д.м.г)....................             |
|Номер первой запису...............(+)..             |
|Номер первой запису...............(-)..             |
|Код группы полученных документов.(,,)..             |
|Код группы выданных документов...(,,)..             |
|Полученные/выданные документы...(+/-)..             |
|Процент для реестра перер-ющего п-ва...             |
------------------------------------------------------

F1 помощь F2/+ расчёт F3 группа F4 очистить F5 настройка F10 выход

F1  - получение этой подсказки;
F2  - начать расчёт реестра налоговых накладных;
+   - начать расчёт реестра налоговых накладных;
F3  - просмотр списка групп видов налоговых накладных;
F4  - очистить меню от введённой информации;
F5  - работа с настройками;
F10 - выход.

  В процессе расчёта программа в домашнем каталоге оператора создаёт каталог
"iceB/nal_nak_xml". В этот каталог записываются налоговые накладные в
формате "*.xml". В начале расчёта программа удаляет из этого каталога 
все файлы "*.xml". После этого записывает налоговые накладные, каждую в
отдельный файл.
  Выгружатся накладные будут только для записей в реестре, которые были
импортированы из бух-учёта. Для записей, которые были введены в реестр
вручную накладные выгружены не будут, так как взять их неоткуда.
  Программа нумерует записи в распечатке начиная с единицы. Можно задать
начало нумерации в реестре введя нужный номер в реквизит меню "Номер первой
записи (+)" для реестра полученных налоговых накладных и "Номер первой
записи (-)" для реестра выданных налоговых накладных.
  Если реестры нужно здавать не целиком, то можно присвоить записям в
реестре код группы. Потом можно задать код группы по которому нужно получить
реестр.
  Программа делает сразу два реестра для полученных и выданных налоговых
накладных. Если нужно один из них, то нужно ввесли в реквизит меню
"Полученные/выданные документы (+/-)" плюс для полученных и минус для
выданных налоговых накладных.
  Если операток ввел "Процент для реестра перер-ющего п-ва", то программа сделает два 
реестра налоговых накладных и додаток 2 для перерабатывающих предприятий. 
  В результате расчёта кроме распечаток будут сформированы отчёты в
електронном виде в формате xml и dbf. Данные в формате dbf могут быть
загружены в систему "Бест Звит".
