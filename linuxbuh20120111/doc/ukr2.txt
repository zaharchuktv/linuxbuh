                    Работа с документами.

	После входа в этот режим работы, на экране появиться меню:

------------------------------------------
|Ввод нового документа.                  |
|Поиск введеного документа.              |
|Просмотр всех введеных документов.      |
|Просмотр документов начиная с дд.мм.гггг|
|Установить/снять блокировку дат.        |
|Выход.                                  |
------------------------------------------

	Режим "Ввод нового документа" позволяет ввести новый документ в
базу данных.
	Режим "Поиск введенного документа" позволяет найти документ по
номеру документа.
	Режим "Просмотр всех введенных документов" позволяет просматривать
список документов введенных в базу данных и просматривать
содержимое записей любого документа из списка найденных документов.
	Режим "Просмотр документов начиная с дд.мм.ггг" позволяет получить
список документов по текущей дате.
	Режим "Установить/снять блокировку дат" позволяет установить/снять
блокировку корректировки документов за любой месяц.
