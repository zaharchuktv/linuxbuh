        Ввод и корректировка кодов целевого назначения.

  Этот режим позволяет ввести список кодов целевого назначения,
которые будут использоваться для получения кассовой книги по
кодам целевого назначения.
  После входа в этот режим на экране появится список кодов 
или сообщение об отсутствии таковых. В ниэней части экрана 
появится строка с подсказкой:

F1 помощь F2 запись F3 удалить F4 поиск F5 печать F6 код F10 выход

F1  - поуление этого текста;
F2  - ввод новой записи;
F3  - удалить запись на которой установлена подсветка строки;
F4  - поиск записи по наименованию;
F5  - распечатка записей;
F6  - поиск записис по коду целевого назначения;
F10 - выход.
