                 Загрузка отработанного времени
                           из файла.
 Этот режим позволяет загрузить отработанное время из файла.
После входа в этот режим на экране появиться запрос:

-------------------
|Введите имя файла|
|                 |
-------------------
F1 помощь F2/+ загрузка F3 просмотр F10 выход

F1  - помощь;
F2  - загрузить файл;
F3  - просмотр и редактирование файла;
F4  - выполнить команды настроенные по клавише F5;
F5  - настроить выполнение нужных команд;
F10 - выход


  Если вам нужно перед загрузкой отработанного времени скопировать файл
в домашний каталог, то соответствующие команды вы можете ввести в
настроечный файл нажав клавишу F5. Введённые оператором комады будут
выполнены после нажатия клавиши F4.



Структура записей в загрузочном файле:


123|1|22|176|31|0|0|
 |  | |   |  |  | |
 |  | |   |  |  | -> Сколько часов в рабочем дне (если не стандартно)
 |  | |   |  |  -> Количество рабочих дней в месяце (если не стандартно)
 |  | |   |  ---> Календарных дней
 |  | |   ---> Часов
 |  | ----> Рабочих дней
 |  --> Код табеля
 ----> Табельный номер
