                Распечатать свод отработанного времени
                          по категориям. 

	После входа в этот режим на экране появиться меню:

-----------------------------------
|Дата начала.......(м.г).         |
|Дата конца........(м.г).         |
|Код подразделения..(,,).         |
|Код категории......(,,).         |
|Код табеля.........(,,).         |
|Табельный номер....(,,).         |
-----------------------------------

F1 помощь F2/+ расчет F3 реквизиты F4 очистить F5 ком.расч. F10 выход

F1  - просмотр документации;
F2  - начать расчет;
+   - начать расчет;
F3  - просмотр реквизитов для заполения меню;
F4  - очистить меню от введенных реквизитов;
F5  - комлексный расчет (расчет используя настройки из файла);
F10 - выход.

	Обязательным для заполнения является только первый
реквизит. Расчет выполняется по любой комбинации введенных
реквизитов. Если дата конца не введена, то программа
устанавливает ее равной дате введенной в первом реквизите.
Тоесть расчет будет выполнен за один месяц. Если код
подразделения не введен, то ведомости будут расчитаны для всех
подразделений. По такомуже принципу программа работает и с
остальными реквизитами. Символы (,,) напротив реквизита
обозначают возможность введения нескольких кодов через запятую.
	Для заполения меню оператор может нажать клавишу F3 и
выбрав список нужного ему реквизита войти в режим просмотра
выбранного списка. Если оператор находясь в списке нажмет
клавишу Enter, то программа вставит код на котором была подсветка
строки в меню.
	Если оператор войдет в режим "Просмотр списка настроек для
комплексного расчета." программа запустит текстовой редактор и
вы попадете в файл который должен иметь вид:

Заводоуправление:
Подразделения|1,2,3
Категории|
Начисления|
Удержания|

Аттестованные:
Подразделения|
Категории|3,5
Начисления|2
Удержания|

	После наименования расчета, который оканчиваеться
двоеточием идет список реквизитов для настройки расчета.
Количетсво наименований расчета не ограничено. 
	Расчет по введенным в файл настройкам осуществляется
нажатием клавиши F5, при этом все реквизиты меню, кроме первых
двух игнорируются.
