               Распечатать отчеты по работникам.
                 
	После входа в этот режим на экране появится меню:


---------------------------------------------------------------------------
|Распечатать список работников.                                           |
|Распечатать справки о зарплате.                                          |
|Распечатать список работников, которые имеют начисления или удержания.   |
|Распечатать список работников, которые не имеют начисления или удержания.|
|Распечатать свод отработанного времени по работникам.                    |
|Распечатать свод начислений и удержаний по работникам.                   |
|Распечатать перерасчет подоходного налога по году.                       |
|Распечатать справку по совместителям.                                    |
|Распечатать долги предприятия перед работниками.                         |
|Распечатать долги работников за месяц.                                   |
|Распечатать свод начислений по работникам                                |
|Распечатать распределение зарплаты мужчины/женщины                       |
|Распечатать карточки начислений/удержаний по работникам.                 |
|Распечатать список работников, которым начислена благотворительная помощь|
|Расчёт общей суммы доходов.                                              |
|Расчёт премии.                                                           |
|Перерасчёт налогов.                                                      |
|Выход.                                                                   |
---------------------------------------------------------------------------

	Режим "Распечатать список работников." позволяет
получить распечатки всех работающих, уволеных, работающих по
совместительству.
	Режим "Распечатать справки о зарплате." позволяет
получить для любого работника справку о зарплате за любой период
времени.
	Режим "Распечатать список работников, которые имеют начисления или
удержания." позволяет получить список работников имеющих интересующее
оператора начисление или удержание.
	Режим "Распечатать список работников, которые не имеют начисления или
удержания." позволяет получить список работников не имеющих интересующее
оператора начисление или удержание.
	Режим "Распечатать свод отработанного времени по
работникам." позволяет получить свод отработанного времени
работниками за любой период времени.
	Режим "Распечатать свод начислений и удержаний по
работникам." позволяет получить свод начислений и удержаний по
работникам за любой период времени с расчетом начального и
конечного сальдо.
	Режим "Распечатать перерасчет налога по году." позволяет
выполнить перерасчет подоходного налога за год для работника с
распечаткой результатов перерасчета.
	Режим "Распечатать справку по совместителям." позволяет
получить справки по совместителям и работающим по договору для
предоставления их в налоговую инспекцию в соответствии с
законодательством Украины.

	Режим "Распечатать долги предприятия перед работниками."
позволяет определить задолженность предприятия по выплате
заработной платы, если на предприятии зарплата выплачивается не
вовремя.
	Режим "Распечатать долги работников за месяц." позволяет
получить распечатку работников у которых по результатам расчета
зарплаты получилось отрицательное сальдо.
	Режим "Распечатать свод начислений по работникам." позволяет
получить по каждому работнику список начиселений за месяц.
	Режим "Распределение зарплаты мужчины/женщины" позволяет получить
свод начисленной зарплаты и отработанного времени для мужчин и женщин.
        Режим "Распечатать карточки начислений/удержаний по работникам"
позволяет получить по работнику свод начислений/удержаний за любой период
времени.
        Режим "Распечатать список работников, которым начислена
благотворительная помощь" позволяет получить список работников, которым
начисленна благотворительная помощь за любой период времени.
	Режим "Расчёт общей суммы доходов" позволяет получить за любой
период времени суммы начисленных доходов в по зарплате и по командировочным
затратам.
	Режим "Расчёт премии" позволяет задать процент для начисления 
премии от начисленных сумм за любой период времени.
	Режим "Перерасчёт налогов" позволяет сделать перерасчёт за месяц
налогов, которые были посчитаны с неправильными настройками.
