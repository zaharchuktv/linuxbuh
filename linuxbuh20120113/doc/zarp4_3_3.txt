                 Распечатать список работников, 
           которые имеют начисления или удержания.


	Этот режим позволяет получить список работников которым
было сделано интересующее оператора начисление или удержание.
	После входа в этот режим на экране появиться меню:

-------------------------------------
|Дата начала.....(м.г)..            |
|Дата конца......(м.г)..            |
|Код подразделения.(,,).            |
|Табельний номер...(,,).            |
|Код начисления....(,,).            |
|Код удержания.....(,,).            |
|Дата нач./удер..(м.г)..            |
|Нач-ния/уде-ния.(+/-)..            |
-------------------------------------

F1 помощь F2/+ расчет F3 реквизиты F4 очистить F10 выход

F1  - помощь;
F2  - расчет;
+   - расчет;
F3  - просмотр реквизитов для заполения меню;
F4  - очистить меню;
F10 - выход.

	Обязательным для заполнения является только первый
реквизит. Если реквизит "Дата конца" не введен, то программа
примет его равным первому реквизиту. Тоесть расчет будет
выполнен за один месяц.
	Расчет выполняется по любой комбинации введенных
реквизитов. Если код подразделения не введен, то расчет будет
выполнен для всех подразделений. По такомуже принципу программа
работает и с остальными реквизитами.
	Для заполнения меню оператор может нажать клавишу F3 и
выбрав список нужного ему реквизита войти в режим просмотра
выбранного списка. Если оператор находясь в списке нажмет
клавишу Enter, то программа вставит код на котором была подсветка
строки в меню.
	Реквизит "Дата нач./удер" позволяет получить начисления
или удержания введенные по заданной дате. При вводе начислений и
удержаний, вводится дата за какой месяц начисление/удержание.
