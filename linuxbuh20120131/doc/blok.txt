       Режим установки/снятия блокировки подсистемы

 Этот режим позволяем устанавливать/снимать блокирование подсистемы 
за нужный месяц или год. Если месяц заблокирован то никто неможет 
выполнять операции ввода, удаления или корректировки данных за указанный
месяц. Доступна только операция чтения. Устанавливать/снимать блокировку
могут только операторы, которым это разрешено. Разрешение это 
устанавливается/снимается с помощью программы i_admint.
 После входа в этот режим на экране вы увидите список заблокированных
дат или сообщение об отсутствии таковых. В нижней части экрана будет
строка подсказки:

F1 помощь F2/+ запись F3 удалить F5 печать F10 выход

F1  - получить эту подсказку
F2  - ввести новую запись
+   - ввести новую запись
F3  - удалить запись на которой установлена подсветка строки
F5  - распечатать список заблокированных дат
F10 - выход
 

 Если нужно разблокировать подсистему для определённог оператора,
то нужно ввести его логин в поле "Логины операторов для которых
разблокировано". В это поле можно ввести при необходимости несколько
логинов через запятую.
 Для подсистемы "Главная книга" предусмотрена возможность блокировать
только операции по введённым в поле "Заблокированные счета" счетам.
В это поле можно ввести при необходимости несколько счетов разделённых
запятыми.
 Если нужно заблокировать целый год, то дату блокировки нужно ввети
так *.год
