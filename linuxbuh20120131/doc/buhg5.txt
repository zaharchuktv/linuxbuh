            Меню "Распечатать отчеты."

        Меню режима имеет следующий вид:
-----------------------------------------------------------
|Расчет оборотного баланса и главной книги                |
|Расчет журнал-ордера по счету                            |
|Расчет сальдо по всем контрагентам                       |
|Расчет сальдо по определенному контрагенту               |
|Распечатать список годов за которые введена информация   |
|Расчет оборотного баланса по внебалансовым счетам        |
|Распечатка документов для перечисления на карт-счета     |
|Отчёт по коментариям                                     |
|Расчёт шахматки                                          |
|Расчёт по заданным спискам дебетовых и кредитовых счетов |
|Расчёт распределения административных затрат на доходы   |
|Расчёт видов затрат по элементам                         |
|Расчёт по спискам групп контрагентов                     |
|Расчёт валовых доходов и затрат                          |
|Расчёт налоговых разниц                                  |
|Расчёт акта сверки по группе контрагентов                |
|Проверка согласованности данных в подсистемах            |
|Выход.                                                   |
-----------------------------------------------------------

	Режим "Расчет оборотного баланса и главной книги."
позволят получит распечатки за любой период времени.
	Режим "Расчет журнал-ордера по счету." позволяет получит
распечатки по любому счету из плана счетов.
	Режим "Расчет сальдо по всем контрагентам." позволяет
получить сальдо по всем контрагентам на любую дату, оборот по
контрагентам за любой период.
	Режим "Расчет сальдо по определенному контрагенту."
позволяет поличить сальдо на любую дату по любому контрагенту.
	Режим "Распечатать список годов за которые введена
информация." позволяет определить за какие годы имеется в базе
данных информация.
	Режим "Расчет оборотного баланса по внебалансовым
счетам." позволяет получить распечатки за любой период времени.
	Режим "Отчёт по коментариям." позволяет получить сводный отчёт за
любой период времени по стандартным коментариям
	Режим "Расчёт шахматки" позволяет расчитать шахматку по списку
счетов введённому оператором.
	Режим "Расчёт по заданным спискам дебетовых и кредитовых счетов"
позволяет по введённому оператором списку дебетовых и кредитовых счетов
получить расчёт шахматки.
        Режим "Расчет распределения административных затрах на доходы."
позволяет получить сумму затрат на рубль полученных доходов.
	Режим "Расчёт видов затрат по элементам." позволяет настроить 
расчёт шахматки в которой по вертикали будут виды затрат, а по 
горизонтали елементы (например подразделения организации).
	Режим "Расчёт по спискам групп контрагентов." позволяет группы
контрагентов объединить в любое количесво списко и по этим спискам
выполнять расчёт журналов-ордеров для счетов с развёрнутым сальдо.
	Режим "Расчёт валовых доходов и затрат." позволяет получить 
расчёт по валовым затратам и доходам.
        Режим "Расчёт акта сверки по группе контрагентов" позволяет 
вычислить общее сальдо по всем контрагентам принадлежащим к группе.
        Режим "Проверка согласованности данных в подсистемах" позволяет
сравнить проводки проведённые в подситеме "Глаыная книга" с данными
в подсистемах "Материальный учёт", "Учёт основных средств".
