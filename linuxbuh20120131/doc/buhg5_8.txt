      Расчёт по списку дебетовых и кредитовых счетов.

 Этот режим позволяет за любой период времени расчитать шахматку по
введённому оператором списку дебетовых и кредитовых счетов. После входа в
этот режим на экране появиться меню:

---------------------------------------
|Дата начала (д.м.г)....              |
|Дата конца  (д.м.г)....              |
|Вариант расчёта (,,)...              |
---------------------------------------

F1 помощь F2/+ расчёт F4 очистить F5 настройка F10 выход

F1  - помощь;
F2  - начать рассчёт;
+   - начать расчёт;
F4  - очистить меню от введённой инфорамации;
F5  - переход в режим редактирования файла с настройками для расчёта;
F10 - выход.


  Все настройки для этого расчёта делаются в файле настройки в который
оператор попадает по нажатию клавиши F5. В этом файле можно настроить любое
количество вариантов расчёта. Каждый вариант расчёта имеет свой код который
можно указать в реквизите меню "Варианты расчёта". Если код варианта расчёта
не введён, то программа сделает расчёт по всем вариантам расчёта которые
есть в файле.
