                Ввод и корректировка расходов на командировки.

	После входа в этот режим на экране появиться список расходов
на командировки или сообщение об отсутствии таковых. в нижней части экрана
буде строка подсказки:

F1 помощь F2/+ запись F3 удалить F4 поиск F5 печать F6 код F9 груп.F10 выход

F1       - просмотр описания режима;
F2       - ввод новой записи;
+        - ввод новой записи;
Shift+F2 - корректировка записи на которой установлена подсветка строки;
F3       - удалить запись на которой установлена подсветка строки;
F4       - поиск записуй по нескольким буквам из наименования;
F5       - распечатка записей;
F6       - поиск записи по коду;
F9       - поиск записей соттветствующей группы вида командировочных
           расходов;
F10      - выход.
