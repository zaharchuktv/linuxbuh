                Распечатка отчетов.

	После входа в этот режим на экране появится меню:

---------------------------------------------------------
|Распечатать движение услуг.                            |
|Распечатать список документов.                         |
|Распечатать реестр проводок по видам операций.         |
|Распечатать список годов за которые введена информация.|
|Выход.                                                 |
---------------------------------------------------------

	Режим "Распечатать движение услуг." позволяет
распечатать все услуги полученные/оказанные за любой период
времени.
	Режим "Распечатать список документов." позволяет
распечатать список документов за любой период времени с
сортировкой по датам или по номерам возростания документов.
	Режим "Распечатать реестр проводок по видам операций."
позволяет распечатать реестр проводок выполненых к введенным
документам по операциям за любой период времени отсортированный
по кодам контрагентов.
        Режим "Распечатать список годов за которые введена
информация." позволяет получить информацию за сколько лет
введена информация в базу данных. Это необходимо для принятия
решения за какие годы уже можно удалть данные с целью уменьшения
объема базы данных.
