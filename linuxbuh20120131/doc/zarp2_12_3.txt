       Импорт начислений/удержаний в документ.

  Этот режим позволяет импортировать в документ начисления из файла. После
входа в этот режим на экране появиться меню:

-------------------
|Введите имя файла|
|                 |
-------------------
F1 помощь F2/+ загруз. F3 просмотр F10 выход

F1  - помощь;
+   - загрузить файл;
F2  - загрузить файл;
F3  - просмотр файла;
F10 - выход.

  Формат записей в файле должен иметь следующий вид:

234|443.24|за секретность|
 |     |       |             
 |     |       ->коментарий             
 |     -> сумма начисления/удержания
 -> табельный номер
