                     Распечатать отчеты.

	После входа в этот режим на экране появится меню:

-----------------------------------------------------------
|Распечатать ведомость зароботной платы.                  |
|Распечатать свод по категориям.                          |
|Распечатать отчеты по работникам.                        |
|Распечатать реестр почтовых переводов алиментов.         |
|Распечатать отчеты по балансовым счетам.                 |
|Распечатать заготовку для заполнения табеля.             |
|Распечатать своды отчислений в гос. фонды.               |
|Распечатать свод начислений и удержаний по подразделению.|
|Распечатать список годов за которые введена информация.  |
|Распечатать свод начислений и удержаний.                 |
|Выход.                                                   |
-----------------------------------------------------------

	Режим "Распечатать ведомость зароботной платы."
позволяет получить ведомости на выдачу зарплаты за любой месяц
по любому подразделению или по произвольно введенному списку
табельных номеров.

        Режим "Распечатать свод по категориям." позволяет
получить свод начислений и удержаний по категориям, свод
начислений/счет и удержаний/счет по категориям, свод
отработанного времени по категориям. Все своды можно получить по
организации, по любому подразделению, по любой комбинации
подразделений, любому начислению или удержанию, любой комбинации
начислений и удержаний.

	Режим "Распечатать отчеты по работникам" позволяет
получить:

	список работников;
	справки о зарплате;
	список работников, которые имеют начисления или удержания;
	список работников, которые не имеют начисления илиудержания;
	перерасчет налога по году;
	свод отработанного времени по работникам;
	свод начислений и удержаний по работникам.

	Режим "Распечатать реестр почтовых переводов алиментов."
позволяет получить реестр почтовых переводов для работников
выплачивающих алименты.

	Режим "Распечатать отчеты по балансовым счетам."
позволяет получить отчеты:
	ведомость производственных затрат;
	свод начислений и удержаний по счетам.

	Режим "Распечатать заготовку для заполнения табеля."
позволяет получить распечатку всех работников по любому
подразделению в виде ведомости для отметки табельшиком
отработанных дней.
	Режим "Распечатать форму N1дф." позволяет получить
распечатку формы 1дф с записью этой формы на дискету для сдачи в
налоговую инспекцию.
	Режим "Распечатать спектр распределения зарплаты."
позволяет задать любой спектр распределения зарплаты для
получения распределения зарплаты в диапазонах заданного спектра.
Если оператор ввел одну цифру 150, то это значит что его
интересует у кого и какая сумма начислена до 150 и у кого свыше
150. Если оператор ввел через запятую несколько цифр, то
программа расчитает у кого и сколько начислено до начала
спектра, внути спектра и больше последней цифры заданного спектра.
	Режим "Распечатать справки для Пенсионного фонда
России." позволяет получить справки для Пенсионного фонда
России с записью их на дискеты в ASCII файлы.
	Режим "Распечатать свод отчислений в социальные фонды."
позволяет получить свод отчислений в социальные фонды, которые
предприятие обязано перечислять.

	Режим "Распечатать свод начислений и удержаний по
подразделению." позволяет получить свод начислений и удержаний
по подразделению. При вводе начислений можно указать код
подразделения, в котором произошло это начисление. И этот свод
будет сформирован на основании кодов подразделений указанных в
начислениях. Код подразделения по умолчанию программой
заполняеться кодом подразделения в котором числится работник.
Если работник работает по совместительству в другом
подразделении то при вводе суммы, которую ему начислили в другом
подразделении нужно ввести код подразделения, за работу в котором
была начисленна эта сумма. Тогда можно получить свод начислений
и удержаний по подразделению не зависимо от того в каких
подразделениях числяться работники получившие начисление за
работу в конкретном подразделении.
 	Режим "Распечатать список годов за которые введена
информация." позволяет определить за какие годы введена
информация о зарплате. Данные о зарплате должны храниться на
предприятии 75 лет.
