        Режим работы с файлами.

 После входа в этот режим на экране появиться список наименований отчётов и
имён файлов в которых эти отчёты находятся. В первом поле строки
отображается метка сохранения файла в виде символа "*" если она выставлена с
помощью клавиши F9. Во втором поле отображается наименование распечатки. В
третьем поле имя файла в котором находится эта распечатка. В третьем поле
размер файла с распечаткой.

 В нихней части экрана появится строка подсказки:

F1 помощь F2 просмотр F3 редактирование F4 коприрование F5 печать F6 очередь
F7 настройка F9 выход F10 выход

F1       - получение этой подсказки;
F2       - просмотр выбранного файла;
Shift+F2 - просмотр выбранного файла перед печатью;
F3       - редактирование файла;
F4       - копирование выбранного файла на DOS дискету, на Windows машину;
F5       - распечатка отчёта, смена принтера;
F6       - просмотр очереди на печать, удаление с очереди на печать;
F7       - включить/выключить просмотр настроек;
F8       - перекодировать выбранный файл в кодировку WINDOWS-1251,
           повторное нажатие клавиши возвращает предыдущюю кодировку;
F9       - выставить/снять метку сохранения файла после выхода по F10;
F10      - выход с удалением всех отчётов;
Tab      - поменять местами коментарий и имя файла;
Enter    - просморт выбранного файла.


  Настройка для передачи файла на другуой WINDOUS-компъютер по сети.
Вся настройка для этого режима должна находится в файле
".linuxbuh/spis_komp.alx", который должен находиться в домашнем каталоге
оператора.  Пример этого файла:

                Файл для настройки передачи распечаток 
                на другие машины по сети.

Главный бухгалтер|nat|iceb_dok|parol|bankcp
Заместитель      |zam|iceb_dok|parol|bankcp
       |           |     |       |     |
       |           |     |       |     -> имя программы, которая будет копировать 
       |           |     |       -> пароль для доступа к каталогу (вводится если для доступа к сетевому диску назначен пароль)
       |           |     -> имя каталога куда нужно записать файл       
       |           -> имя машины на которую нужно записать файл
       -> Коментарий                        


	Настроить можно любое количество машин на которые можно копировать файлы. 

Программа копирования запуститься со следуюшими ключами:

bankcp nat  imaf imaf iceb_dok parol
  |     |     |   |       |     |
  |     |     |   |       |     -> пароль (вводится если для доступа к сетевому диску назначен пароль)
  |     |     |   |       -> имя сетевого диска
  |     |     |   -> под каким именем записать пересылаемый файл
  |     |     ->имя файла который пересылается
  |     -> имя машины куда нужно переслать файл
  -> имя программы, которая осуществляет передачу файла


                     Настройка выполнения программы.
 Режим "Выполнить программу" позволяет настроить запуск любой программы с
именем выбранного файла в списке параметров. Тоесть вы можете написать
программу или скрипт который будет выполнять нужные вам действия с файлом.
Например передать файл по сети на другую машину. 
Настройки запуска нужных вам программ выполняются в файле
".linuxbuh/progforlst.alx", который должен находиться в домашнем каталоге
оператора. Пример этого файла:

Передать файл по сети главбуху|sendglb
Отправить файл по e-mail|sendemail
         |                 |                   
         |                 -> имя программы которая будет выполнять нужую
         |                    операцию
         --> коментарий (появится в меню для выбора если в файле будет больше
             чем одна настройка)
