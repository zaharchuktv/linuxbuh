                     Работа с журналом платёжных ведомостей.

     Этот режим предназначен для регистрации платёжных ведомостей в журнале
с целью автоматического присваивания следующего порядкового номера новой
ведомости.
	После входа в этот режим на экране появится список
зарегистрированных ведомостей или сообщение об отсутствии таковых. В нижней
части экрана расположена строка подсказки:

F1 помощь F2 запись F3 удалить F4 поиск F5 печать F10 выход

F1  - получение подсказки;
F2  - ввод новой записи;
F3  - удаление записи на которой установлена подсветка строки;
F4  - задание реквизитов поиска записей;
F5  - распечатать;
F10 - выход.

  Файл для загрузки должен состоять из строк описывающих шапку документа и содержимое документа.
Первой строкой должна быть строка с описанием шапки документа. После неё может идти любое 
количество строк с содержимым документа. В файле может быть любое количество документов.


Формат шапки кассового ордера:
KDOK|-|1|27.3.2007|1-1|301|пд|основание|додаток|призвище|документ|
 |   |      |       |   |  |     |         |       |        |
 |   |      |       |   |  |     |         |       |        -> реквизиты документа удостоверяющено личность
 |   |      |       |   |  |     |         |       -> фамилия получившего/внесшего деньги
 |   |      |       |   |  |     |         -> дополнение
 |   |      |       |   |  |     -> основание
 |   |      |       |   |  -> код операции
 |   |      |       |   -> счет
 |   |      |       ->номер документа (если вместо номера документа пробел - программа сама присвоит номер)
 |   |      -> дата документа
 |   -> "+" - приход "-" - расход
 -> код строки




Формат записи кассового ордера:

KZAP|тн5|150.3|БОНДАРЧУК В.П.|
 |    |    |        |
 |    |    |        -> фамилия (необязательный реквизит, при загрузке не используется)
 |    |    -> сумма
 |    -> код контрагента
 -> код строки
