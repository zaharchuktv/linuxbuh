            Работа с файлом настройки.

        В файле настройки описаны параметры, которые пользователь
может менять по своему усмотрению. При входе в этот режим
программа запускает тестовой редактор с помощью которого вы
можете менять те или иные параметры. Правила работы в редакторе
зависят от типа редактора используемого вами. Какой редактор
будет запускаться для работы с файлом настройки указывается в
самом файле настройки.

        Имя файла matnast.alx и содержимое файла имеет вид:

Исполнитель|ииии
Округление 1|0.01
Округление цены|0.000001
Н.Д.С.|20
Стартовий год|1999

Перечень материальных счетов|05,06,07,08,12,40,41
Счета розничной торговли|41/2

Раздельная нумерация накладных|Включена
Код операции торговли|ПП,РП,РКО,РР1,РР2

Цена материала с НДС|Включена
Наценка на справочную цену|Выключена

Перенос символа N|Выключено
Код тары|
Код группы возвратная тара|
Наименование операции в накладной|Включено
Корректор межстрочного растояния|0
Отступ от левого края|5
Отступ счет-фактуры|5
Счет организаций|61
Сортировка записей в накладной|Выключена
Отчет в килограммах|Выключено
0% НДС реализация на територии России|
0% НДС экспорт|
0% Освобождение от НДС|
Объединение одинаковых материалов в накладной|Включено
Курс УЕ|31.00
Группы для склада 1|
Отпуск разрешил|
Верхний отступ для накладной М-20|
Левый отступ для накладной М-20|
Отпустил 1|
Контроль остатка на выписке|Включено

        Все наименования параметров отделены от параметра
символом "|".
        Параметр "Н.Д.С." указывает на величину налога на
добавленную стоимость в процентах.
	Параметр "Стартовый год" указывет в каком году были
введены стартовые остатки. Программа при просмотре документов и
вычислении остатков начинает поиск данных с этого года. Для
увеличения быстродействия системы предусмотрен режим переноса
остатков на следующий год (программа saldo). После этого надо
установить стартовую дату на год на который были перенесены
стартовые остатки.
        Параметр "Перечень материальных счетов" содержит список
всех материальных счетов из плана счетов.
        Если параметр "Раздельная нумерация накладных" включен,
то программа будет вести нумерацию для приходов и расходов
раздельно. Если этот параметр выключен то при выписке документов
будет предлагаться сквозная нумеразия документов для прихода и
расхода.
        В параметре "Код операции торговли" указываются через
запятую коды операций для которых необходимо, чтобы в распечатке
документа была колонка в которой былабы указана цена с НДС.
        Если параметр "Цена материала с НДС" включен, то
программа считает, что все справочные цены введенные в базу
данных введены с НДС. Если параметр выключен, то програма
считает что все справочные цены введены без НДС.
        Если параметр "Наценка на справочную цену" включен, то
программа при расчете цены с использованием процента наценки
будет брать справочную цену введенную в справочнике материалов.
Если параметр выключен, то программа для расчета цены с
использованием процента наценки будет брать цену учета материала
(цену в карточке материала).
        Если парамет "Перенос символа N" включен, и если номер
документа имеет вид 1-123, программа в распечатке документа
поставит символ N в середину: 1-N123. Если параметр выключен
программа напечатает так: N1-123.
        В параметре "Код тары" через запятую могут быть указаны
коды тары. Это приведет к тому что в распечатке накладной тара
будет распечатана внизу документа, отдельно от остальных
материалов.
        В параметре "Код группы возвратная тара" может быть
указан код группы материалов, которая объединяет всю возвратную
тару. На эту тару не будет начислятся НДС и она будет в
распечатке документа напечатана отдельно от остальных
материалов. 
        Если параметр "Наименование операции в накладной"
включен, то в распечатке накладной будет напечатано наименование
операции.
        Параметр "Корректор межстрочного растояния" используется
для установки нужного межстрочного расстояния при печати
накладных.
        Параметр "Отступ от левого края" устанавливает величину
поля с левой стороны распечатки для печати накладных.
        Параметр "Счет организаций" сообщает программе на каком
счету ведется расчеты с контрагентами.
        Если параметр "Сортировка записей в накладной" включен,
то в распечатке документа записи будут отсортированы по группам,
а внутри группы по наименованию.
        Если параметр "Отчет в килограммах" включен то в
распечатке движения тов.-матер. ценностей (форма 1), кроме
движения в единицах измерения будет строка где такоеже движение
показано в пересчете на киллограмы.
        В перечисленных ниже трех параметрах указываются
соответствующие коды операций:
0% НДС реализация на територии России;
0% НДС экспорт;
0% Освобождение от НДС.
        Если параметр "Объединение одинаковых материалов в
накладной" включен, то одинаковые записи материалов в документе
при распечатке будут объединены в одну запись.
        Параметр "Курс УЕ" сообщает программе какой курс валюты
нужно использовать при формировании прайс листа.
