      Работа с реестром полученных счет-фактур.

  После входа в этот режим на экране появиться список накладных
или сообщение об отсутствии таковых. В нижней части экрана
появиться строка подсказки.

F1 помощь F2/+ запись F3 удалить F4 поиск F5 печать F6 импорт F10 выход

F1       - помощь;
F2       - ввести новую запись в реестр;
+        - ввести новую запись в реестр;
F3       - удалить запись на которой установлена подсветка
           строки;
Shift+F3 - удалить все записи из реестра с учётом условий
           поиска;
F4       - поиск нужных записей;
F5       - получить распечатку реестра счет-фактур;
F6       - импорт документов из подсистем бухгалтерского учёта;
F10      - выход.

  Импорт документов по клавише F6 осуществляется из подситем:

  Главная книга;
  Материальный учёт;
  Учёт услуг;
  Учёт основных средств;
  Учёт командировочных расходов.

  Из подсистемы "Главная книга" импорт осуществляется если по cчёту 
указанному в файле настройки есть проводки с коментарием следующего вида:

нн55 1.1.2008 ПН

 Где:
 "нн"     - обязательный идентефикатор;
 "55"     - номер счет-фактуры;
 1.1.2008 - дата выписки счет-фактуры.
 ПН       - вид документа 

 Проводка должна быть сделана на сумму НДС. Во время импорта программа
основывываясь на сумме НДС, вычислит сумму по документу. Дата проводки будет
датой получения счет-фактуры. Если проводка сделана по дебету, то она
будет импортирована в реестр полученных счет-фактур, если по
кредиту, то она будет импортирована в реестр выданных счет-фактур.

 Из подсистем "Материальный учёт", "Учёт услуг", Учёт основных средств", 
будут импортироваться документы в шапке которых введён номер счет-фактуры
 и дата получения счет-фактуры. 

 Для импорта из подсистемы "Учёт командировочных расходов" нужно чтобы в 
документе при вводе командировочных расходов в меню были введены реквизиты:

Код контрагента для реестра счет-фактур;
Номер документа.
