        Импорт карт-счетов 

  Формат файла для импорта карт-счетов должен иметь следюющий вид:

1234567890 0987654321
    |          |
    |          -> номер карт-счёта в банке
    -> Идентифакационный номер работника

  Между идентификационным номером и номером карт-счёта должен быть только
один пробел. После номера картсчёта через пробел может быть любая информация
которая программой будет игнорироваться, но может быть полезной. Например
фамилия человека, которому принадлежит этот карт-счёт.
