                 Получение ведомости на выдачу
                зарплаты по произвольному списку.

	После входа в этот режим на экране появиться меню:

----------------------------------
|Дата расчета...(м.г)..          |
|Имя файла со списком..          |
|Код начисления (,,)...          |
----------------------------------

F1 помощь F2/+ расчет F3 реквизиты F4 очистить F10 выход

F1  - просмотр документации;
F2  - начать расчет;
+   - начать расчет;
F3  - просмотр реквизитов для заполения меню и файла со списком
      табельных номеров для расчета;
F4  - очистить меню от введенных реквизитов;
F10 - выход.

	Обязательным для заполнения является только первый
реквизит. Расчет выполняется по любой комбинации введенных
реквизитов. Если реквизит "Код начисления" введен то в
распечатку попадут только начисления указанные в этом реквизите.
	Для заполения меню оператор может нажать клавишу F3 и
выбрав список нужного ему реквизита войти в режим просмотра
выбранного списка. Если оператор находясь в списке нажмет
клавишу Enter, то программа вставит код на котором была подсветка
строки в меню.
	Если оператор выберет режим "Просмот файла со списком
для расчета." программа запустит текстовой редактор для файла
указанного во втором реквизите меню. В этом файле в столбик
должны быть указаны табельные номера для которых нужно выполнить
расчет.
	После завершения расчета программа сообщит в какой файл
будет выгружена распечатка.
