                  Распечатка справки по совместителям.

	После входа в этот режим на экране появится меню:

-------------------------------
|Дата начала....(м.г)..       |
|Дата конца.....(м.г)..       |
|Табельний номер.(,,)..       |
|Код контрагента (,,)..       |
-------------------------------

F1 помощ F2/+ расчет F3 реквизиты F4 очистить F10 выход

F1  - помощь;
F2  - начать расчет;
+   - начать расчет;
F3  - просмотр списка табельных номеров, контрагентов;
F4  - очистить меню;
F10 - выход.

	Обязательными для заполнения является только два первых
реквизита. 
	Если реквизит "Табельний номер" не введен, то расчет будет
выполнен по всем совместителям. 
        В соответствии с законодательством России справки делаются не
только для людей работающих по совместительству, но и на
всех частных предпринимателей, которым были перечисленны деньги. В
реквизит "Код контрагента" можно ввести код контрагента частного
предпринимателя если справку нужно сделать только для него. Кроме того при
вводе частного предпринимателя в список контрагентов обязательно должен
быть заполнен реквизит "Регистрационный номер частного предпринимателя", в
настроечном файле должен быть введен "Счет учета частных
предприниматедей". При необходимости счетов может
быть несколько и тогда они должны быть введены через запятую.
	Метка совместителя устанавливается в карточке
конкретного работника с помощю клавиши F7. Данные для заполения
справок также беруться из карточки. После входа в карточку
работника нужно нажать клавишу Tab и в открывшемся меню
заполнить все реквизиты.
	После нажания клавиши F2 программа выполнит расчет и
результаты расчета выгрузит в файлы имена которых выведет на
экран.
	Нажатие клавиши F3 позволяет просмотреть список
табельных номеров для поиска нужного табельного номера.
	Расчет выполняется по выплаченной зарплате. Программа смотрит
в счет каких месяцев были сделаны выплаты, определяет полноту
выплаты и формирует справки.
